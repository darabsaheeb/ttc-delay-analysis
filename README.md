# TTC Transit Delay Analysis (2023)

Analysis of Toronto Transit Commission (TTC) delay data for 2023 across subway, streetcar, and bus services.

## Research Questions
- Which TTC routes had the highest total delay time in 2023?
- At what times of day do delays occur most frequently?
- What are the most common causes of TTC delays?
- Are delays more severe on weekdays vs weekends?
- How do delay patterns differ between transit modes?

## Tools Used
Python, Pandas, SQL (SQLite), Matplotlib, Seaborn

## Dataset
Publicly available TTC delay records sourced from Kaggle.
